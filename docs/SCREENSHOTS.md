To showcase Moss effectively, please capture the following screenshots.

**General Guidelines:**
- **Window Size**: ~1400x900 (standard laptop aspect ratio).
- **Theme**: Dark Mode (it looks most premium).
- **Content**: Use "realistic" looking notes (not just "test test").

## 1. The Hero Shot (Editor + Sidebar)
- **View**: Main editor open with a formatted note. Sidebar visible.
- **Content**: A note with Headers, a List, and maybe a Quote.
- **Purpose**: Shows the core writing experience.
- **Filename**: `hero-editor.png`

## 2. The Split (Split Panes)
- **View**: Two notes side-by-side (`Cmd+\`).
- **Content**:
    - Left: A "Research" note.
    - Right: A "Draft" note being written based on the research.
- **Purpose**: Demonstrates multitasking power.
- **Filename**: `feature-split.png`

## 3. The Brain (Graph View)
- **View**: Graph view open (sidebar or main view).
- **Content**: A nice cluster of nodes. Hover over one to show connections if possible.
- **Purpose**: Showcases the "knowledge graph" aspect.
- **Filename**: `feature-graph.png`

## 4. The Intelligence (AI Palette)
- **View**: Text selected, AI Command Palette (`Cmd+K`) open.
- **Content**: "Rewrite this to be more concise" typed in the input.
- **Purpose**: Highlights AI capabilities.
- **Filename**: `feature-ai.png`

## 5. The Empty State (Zero Inbox)
- **View**: No tabs open.
- **Content**: The new "Welcome to Moss" empty state with the logo.
- **Purpose**: Shows the polished onboarding detail.
- **Filename**: `feature-empty.png`
